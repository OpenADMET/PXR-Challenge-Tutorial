# PXR-Challenge-Tutorial
[![Logo](https://img.shields.io/badge/OSMF-OpenADMET-%23002f4a)](https://openadmet.org/)

This repo provides starter notebooks and example workflows for the [**OpenADMET PXR Blind Challenge**](https://huggingface.co/spaces/openadmet/pxr-challenge) — a community benchmark for predicting **PXR (Pregnane X Receptor) induction**.

For full challenge details, background, and dataset descriptions, see the [announcement post](https://openadmet.ghost.io/announcing-the-next-openadmet-blind-challenge-predicting-pxr-induction/).

## 📦 Data

Training and test data are available on Hugging Face:
[`openadmet/pxr-challenge-train-test`](https://huggingface.co/datasets/openadmet/pxr-challenge-train-test)

## 📓 Tutorials

* [**Activity Track**](./notebooks/activity_prediction.ipynb): Training regression models on pEC50 data.
* [**Structure Track**](./notebooks/structure_prediction.ipynb): Structural modeling and submission formatting.
